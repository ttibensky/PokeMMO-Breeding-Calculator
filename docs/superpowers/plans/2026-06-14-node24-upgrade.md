# Node.js 24 Upgrade Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Clear the Node 20 GitHub Actions deprecation across both workflows, move the build/test runtime to Node 24, and pin local dev to Node 24.

**Architecture:** Pure configuration change. Bump every GitHub Action to its Node-24 major, raise `setup-node`'s `node-version` 22→24, bump `@types/node` to match, add `.nvmrc` + `engines.node` to pin local dev, and update the README prerequisite. No application code or test behavior changes — verification is the existing full suite passing on Node 24.

**Tech Stack:** GitHub Actions, Node.js 24 (LTS), npm, Vite 6, Vitest 3, Playwright 1.52, TypeScript 5.8.

**Spec:** `docs/superpowers/specs/2026-06-14-node24-upgrade-design.md`

```json

```

and replace the version with `^24`:


- [ ] **Step 2: Add an `engines` field**

In `package.json`, insert a top-level `"engines"` block immediately after the `"scripts"` block's closing brace (line 16) and before `"dependencies"` (line 17). The region currently reads:
