# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Light / Dark / Auto color-scheme control on the Settings page, using Mantine v7's native color-scheme system.

**Architecture:** Remove the hardcoded `forceColorScheme="light"` lock on `MantineProvider` and set `defaultColorScheme="auto"`. Add an "Appearance" `SegmentedControl` on the Settings page bound to Mantine's `useMantineColorScheme()` hook. Mantine handles persistence (its own localStorage key) and system-preference following for "Auto" automatically — no Zustand or import/export changes.

**Tech Stack:** React 18, TypeScript, Vite, Mantine v7 (`@mantine/core ^7.17.8`), Zustand (unchanged here), Playwright (e2e).

**Note on flash prevention:** The spec mentioned `ColorSchemeScript`. That is an SSR-only mechanism and does not reliably execute inside a Vite CSR React root; the official Mantine Vite template omits it. Because this app's `#root` is empty until the JS bundle loads, there is no pre-render content to flash. `defaultColorScheme="auto"` on `MantineProvider` is the correct, complete approach for this stack — so `ColorSchemeScript` is intentionally not added.

```typescript
  // ── 4. Color scheme (dark mode) can be set and persists ─────────────────────
  test('color scheme can be set to dark and persists across reload', async ({ page }) => {
    await freshStart(page, './#/settings');

    // Select the Dark color scheme in the Appearance segmented control
    await page.getByText('Dark', { exact: true }).click();
    await expect(page.locator('html')).toHaveAttribute(
      'data-mantine-color-scheme',
      'dark'
    );

    // Reload — the preference should persist
    await page.reload();
    await expect(page.getByRole('heading', { name: 'Settings' })).toBeVisible();
    await expect(page.locator('html')).toHaveAttribute(
      'data-mantine-color-scheme',
      'dark'
    );
  });
```
